Introduction to HTML

HTML stands for HyperText Markup Language. It is used to create the structure of web pages. It tells the browser how content should be displayed.

Chapter 1 HTML Structure

HTML structure defines the basic layout of a web page. It includes elements like html, head, and body. Everything written on a webpage goes inside this structure.

Chapter 2 HTML Text

HTML text is used to display written content on a webpage. Headings, paragraphs, and formatting are used for text.

Chapter 3 HTML List

Lists are used to organize items in order or bullet form. There are two types: ordered lists and unordered lists.

Chapter 4 HTML Links

Links are used to connect one page to another. They help users navigate between pages or websites.

Chapter 5 HTML Images

Images are used to display pictures on a webpage using a source file path.

Chapter 6 HTML Tables

Tables are used to display data in rows and columns in a structured way.
